# 题目
1. 已知一个名为Complex的复数类，这个类包含：  
    （1）私有成员：实部、虚部，且均为int 型  
    （2）公有的带默认形参值的构造函数、复制构造函数  
    （3）公有成员函数Display，其作用为显示复数  
    要求：  
    （1）实现满足上述属性和行为的Complex类定义；  
    （2）设计函数AddComplex，函数AddComplex功能为实现两个复数相加，要求该函数的形参为复数类的常引用；  
    （3）保证如下主函数能正确运行。

### 裁判测试程序样例：

```c++

    int x,y;
```

# 解答
```cpp
#include <iostream>
using namespace std;

class Complex {
private:
    int real;  // 实部
    int imag;  // 虚部
public:
    // 带默认参数的构造函数
    Complex(int r = 0, int i = 0) : real(r), imag(i) {}
    // 复制构造函数
    Complex(const Complex &c) : real(c.real), imag(c.imag) {}
    // 显示复数
    void Display() {
        if (imag > 0) {
            cout << real << "+" << imag << "i" << endl;
        } else if (imag < 0) {
            cout << real << imag << "i" << endl;
        } else {
            cout << real << endl;
        }
    }
    // 声明AddComplex为友元函数，使其能访问私有成员
    friend Complex AddComplex(const Complex &c1, const Complex &c2);
};

// 实现复数相加函数
Complex AddComplex(const Complex &c1, const Complex &c2) {
    int newReal = c1.real + c2.real;
    int newImag = c1.imag + c2.imag;
    return Complex(newReal, newImag);
}

```
