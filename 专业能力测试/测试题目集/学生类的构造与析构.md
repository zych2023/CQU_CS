# 题目
### 类定义：

```c++
定义一个学生类Student，使得main()函数能够得到指定的输出结果
```

### main()函数如下：

```c++
/* 请在这里填写答案 */
int main()
  {Student stud1(10010,"Wang_li",'f');
   stud1.display();
   Student stud2(10011,"Zhang_fun",'m');
   stud2.display();
   return 0;
}
```

### 输入样例：

```in
无
```

### 输出样例：

在这里给出相应的输出。例如：

```out
Constructor called.
num:10010
name:Wang_li
sex:f

Constructor called.
num:10011
name:Zhang_fun
sex:m

Destructor called.
Destructor called.
```

# 解答
```cpp
#include <iostream>
#include <string>
using namespace std;

class Student {
private:
    int num;       // 学号
    string name;   // 姓名
    char sex;      // 性别
public:
    // 构造函数，初始化成员变量并输出构造信息
    Student(int n, string s, char c) : num(n), name(s), sex(c) {
        cout << "Constructor called." << endl;
    }
    
    // 显示学生信息
    void display() {
        cout << "num:" << num << endl;
        cout << "name:" << name << endl;
        cout << "sex:" << sex << endl << endl;
    }
    
    // 析构函数，输出析构信息
    ~Student() {
        cout << "Destructor called." << endl;
    }
};
```
