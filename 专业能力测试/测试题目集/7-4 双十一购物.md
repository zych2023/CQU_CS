# 题目
现在的双十一购物活动真是越搞越疯狂吖！哪怕只是你买了9.9元的物品，商家也给你直接邮寄到家—包邮！如果你买的东西价钱高于9.9，那就当然更给你包邮了！  
然而，并不是所有的物品都包邮哦，低于9.9元的是需要额外收取邮费的，不过双十一期间邮费也给广大客户搞活动，不管你邮寄到哪里，一件物品的邮费就要你0.99元。  
Jeremy双十一期间买了三件物品，请你帮忙计算，他本次一共花钱？

### 输入格式:

输入三个实数，分别代表Jeremy双十一期间买的三件物品的价钱。

### 输出格式:

输出他买三件物品以及包括邮费一共花多少钱？  
输出结果保留两位小数  
输出结果后输出回车换行

### 输入样例:

在这里给出一组输入。例如：

```in
9.9 10 6.78
```

### 输出样例:

在这里给出相应的输出。例如：

```out
27.67
```

### 样例解释:

9.9的物品和10的物品不花邮费，只花物品价钱本身，但是6.78不包邮，需要额外0.99的邮费  
所以一共花钱：9.9 + 10 + 6.78 + 0.99 = 27.67

# 解答
```cpp
#include <iostream>

using namespace std;

int main() {
    // 定义三个变量存储三件物品的价格
    double price1, price2, price3;
    // 读取输入的三个实数
    cin >> price1 >> price2 >> price3;
    
    double total = 0.0;  // 总花费初始化为0
    
    // 处理第一件物品：判断是否需加邮费
    if (price1 < 9.9) {
        total += price1 + 0.99;
    } else {
        total += price1;
    }
    
    // 处理第二件物品
    if (price2 < 9.9) {
        total += price2 + 0.99;
    } else {
        total += price2;
    }
    
    // 处理第三件物品
    if (price3 < 9.9) {
        total += price3 + 0.99;
    } else {
        total += price3;
    }
    
    // 输出总花费，保留两位小数
    printf("%.2f",total);
    return 0;
}
```
