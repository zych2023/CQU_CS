# 题目
1元5角钱人民币兑换5分、2分和1分的硬币（每一种都要有）共100枚，会有很多种兑换方案。请编写程序给出各种兑换方案。

### 输入格式:

输入为一个正整数n，表示要求输出前n种可能的方案。方案的顺序，是按照5分硬币从少到多排列的。

### 输出格式:

显示前n种方案中5分、2分、1分硬币各多少枚。每行显示一种方案，数字之间空一格，最后一个数字后没有空格。

注意：如果全部方案不到n种，就顺序输出全部可能的方案。

### 输入样例:

```in
5
```

### 输出样例:

```out
1 46 53  
2 42 56  
3 38 59  
4 34 62  
5 30 65
```

# 解答
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    
    int count = 0;
    // x为5分硬币数量，范围1到12
    for (int x = 1; x <= 12; ++x) {
        int y = 50 - 4 * x; // 2分硬币数量
        int z = 50 + 3 * x; // 1分硬币数量
        
        // 输出当前方案
        cout << x << " " << y << " " << z << endl;
        count++;
        
        // 达到n种方案则退出
        if (count == n) {
            break;
        }
    }
    
    return 0;
}
```
