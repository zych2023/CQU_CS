# 题目

本题要求将输入的任意3个整数从小到大输出。

### 输入格式:

输入在一行中给出3个整数，其间以空格分隔。

### 输出格式:

在一行中将3个整数从小到大输出，其间以“->”相连。

### 输入样例:

```in
4 2 8
```

### 输出样例:

```out
2->4->8
```

# 解答

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b, c;
    cin >> a >> b >> c;
    // 排序：确保a ≤ b ≤ c
    if (a > b) swap(a, b);  // 第一步：让a ≤ b
    if (a > c) swap(a, c);  // 第二步：让a成为最小值
    if (b > c) swap(b, c);  // 第三步：让b ≤ c
    
    // 按格式输出，用"->"连接
    cout << a << "->" << b << "->" << c << endl;
    
    return 0;
}
```
