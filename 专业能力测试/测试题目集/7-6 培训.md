# 题目
某培训机构的学员有如下信息：

- 姓名（字符串）
    
- 年龄（周岁，整数）
    
- 去年 NOIP 成绩（整数，且保证是 5 的倍数）
    

经过为期一年的培训，所有同学的成绩都有所提高，提升了 20%（当然 NOIP 满分是 600 分，不能超过这个得分）。

输入学员信息，请设计一个结构体储存这些学生信息，并设计一个函数模拟培训过程，其参数是这样的结构体类型，返回同样的结构体类型，并输出学员信息。

### 输入格式:

第一行输入一个正整数n，表示学员个数。

第二行开始往下n行。每行首先是一个字符串表示学员姓名，再是一个整数表示学员年龄，再是一个整数为去年NOI绩。

### 输出格式:

输出 n 行，每行首先输出一个字符串表示学生姓名，再往后两个整数，表示经过一年的培训后学员的年龄和他们今年的 NOIP 成绩。空格隔开。  
**说明/提示**

数据保证，1<=n<=20。年龄为 0~100（含 0 与 100）的整数。成绩为 （含 0 与 600）的 5 的整倍数。

### 输入样例:

在这里给出一组输入。例如：

```in
3
kkksc03 24 0
chen_zhe 14 400
nzhtl1477 18 590
```

### 输出样例:

在这里给出相应的输出。例如：

```out
kkksc03 25 0
chen_zhe 15 480
nzhtl1477 19 600
```

# 解答
```cpp
#include <iostream>
#include <string>
#include <vector>

using namespace std;

// 定义学员结构体
struct Student {
    string name;   // 姓名
    int age;       // 年龄
    int score;     // NOIP成绩
};

// 模拟培训过程的函数
Student train(Student s) {
    // 年龄增加1岁
    s.age += 1;
    
    // 成绩提升20%，但不超过600分
    int newScore = s.score * 1.2;
    s.score = (newScore > 600) ? 600 : newScore;
    
    return s;
}

int main() {
    int n;
    cin >> n;
    
    // 存储所有学员信息
    vector<Student> students(n);
    
    // 读取学员信息
    for (int i = 0; i < n; ++i) {
        cin >> students[i].name >> students[i].age >> students[i].score;
    }
    
    // 培训并输出结果
    for (int i = 0; i < n; ++i) {
        Student trainedStudent = train(students[i]);
        cout << trainedStudent.name << " " 
             << trainedStudent.age << " " 
             << trainedStudent.score << endl;
    }
    
    return 0;
}
    
```

    s.noip=s.noip*1.2>600?600:s.noip*1.2;
    cout<<s.name<<" "<<s.age<<" "<<s.noip<<endl;
    return;
}

int main() {
    int n;
    cin>>n;
    for(int i=0;i<n;i++){
        stu s;
        cin>>s.name>>s.age>>s.noip;
        train(s);
    }
    return 0;
}
