# 题目
从键盘输入N（0<=N<=100）个学生的成绩（成绩为整数），计算学生们的平均成绩，并统计及格（成绩不低于60分）的人数。

### 输入格式:

第一行输入学生人数N。从第二行开始给出N个非负整数，即这N位学生的成绩，其间以空格或回车符分隔。

### 输出格式:

average = 平均成绩（保留小数点后2位）  
count = 及格人数

### 输入样例1:

```in
10
65 34 75 68 80 79 92 26 0 55
```

### 输出样例1:

```out
average = 57.40
count = 6
```

### 输入样例2:

```in
0
```

### 输出样例2:

```out
average = 0.00
count = 0
```

# 解答
```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int N;
    cin >> N;

    int sum = 0;
    int count = 0;

    for (int i = 0; i < N; ++i) {
        int score;
        cin >> score;
        sum += score;
        if (score >= 60) {
            count++;
        }
    }

    double average;
    if (N == 0) {
        average = 0.0;
    } else {
        // average = static_cast<double>(sum) / N;
        average =(double)sum / N;
    }

    cout << "average = " << fixed << setprecision(2) << average << endl;
    cout << "count = " << count << endl;

    return 0;
}
```
