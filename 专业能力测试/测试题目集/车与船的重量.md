# 题目
定义一boat与car两个类，二者都有weight属性，定义二者的一个友元函数totalweight()，计算二者的重量和。

### 测试程序样例：

```c++
在这里给出函数被调用进行测试的例子。例如：
#include <iostream>
using namespace std;

/* 请在这里填写答案 */

int main()
{
  int c,b;
  cin>>c>>b;
  car c1(c);
  boat b1(b);
  cout<<totalweight(b1,c1)<<endl;
}
```

### 输入样例：

```in
1000 2000
```

### 输出样例：

```out
3000
```
# 解答
```cpp
// 提前声明类，以便友元函数使用
class car;

class boat {
private:
    int weight;
public:
    // 构造函数，初始化重量
    boat(int w) : weight(w) {}
    // 声明totalweight为友元函数
    friend int totalweight(const boat& b, const car& c);
};

class car {
private:
    int weight;
public:
    // 构造函数，初始化重量
    car(int w) : weight(w) {}
    // 声明totalweight为友元函数
    friend int totalweight(const boat& b, const car& c);
};

// 友元函数实现，计算boat和car的重量和
int totalweight(const boat& b, const car& c) {
    return b.weight + c.weight;
}

```
