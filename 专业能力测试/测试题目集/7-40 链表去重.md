# 题目
给定一个带整数键值的链表 L，你需要把其中绝对值重复的键值结点删掉。即对每个键值 K，只有第一个绝对值等于 K 的结点被保留。同时，所有被删除的结点须被保存在另一个链表上。例如给定 L 为 21→-15→-15→-7→15，你需要输出去重后的链表 21→-15→-7，还有被删除的链表 -15→15。

### 输入格式：

输入在第一行给出 L 的第一个结点的地址和一个正整数 N（≤105，为结点总数）。一个结点的地址是非负的 5 位整数，空地址 NULL 用 -1 来表示。

随后 N 行，每行按以下格式描述一个结点：

```
地址 键值 下一个结点
```

其中`地址`是该结点的地址，`键值`是绝对值不超过104的整数，`下一个结点`是下个结点的地址。

### 输出格式：

首先输出去重后的链表，然后输出被删除的链表。每个结点占一行，按输入的格式输出。

### 输入样例：

```in
00100 5
99999 -7 87654
23854 -15 00000
87654 15 -1
00000 -15 99999
00100 21 23854
```

### 输出样例：

```out
00100 21 23854
23854 -15 99999
99999 -7 -1
00000 -15 87654
87654 15 -1
```

# 解答
```cpp
#include <iostream>
#include <cstdio>
#include <cmath>
using namespace std;

// 定义节点结构
struct Node {
    int key;
    int next;
} nodes[100000]; // 节点地址为5位整数，最大地址99999

int main() {
    int head, N;
    cin >> head >> N;

    // 读取所有节点
    for (int i = 0; i < N; ++i) {
        int addr, key, next;
        cin >> addr >> key >> next;
        nodes[addr].key = key;
        nodes[addr].next = next;
    }

    bool exist[10001] = {false}; // 记录键值绝对值是否已出现（键值绝对值≤10^4）
    int new_head = -1, new_tail = -1; // 去重链表的头和尾
    int del_head = -1, del_tail = -1; // 删除链表的头和尾

    int current = head;
    while (current != -1) {
        int key = nodes[current].key;
        int abs_key = abs(key);
        int next_addr = nodes[current].next; // 保存下一个地址

        if (exist[abs_key]) {
            // 键值绝对值已存在，加入删除链表
            if (del_head == -1) {
                del_head = current;
                del_tail = current;
            } else {
                nodes[del_tail].next = current;
                del_tail = current;
            }
            nodes[del_tail].next = -1; // 暂时置为-1，后续可能更新
        } else {
            // 键值绝对值未存在，加入去重链表
            exist[abs_key] = true;
            if (new_head == -1) {
                new_head = current;
                new_tail = current;
            } else {
                nodes[new_tail].next = current;
                new_tail = current;
            }
            nodes[new_tail].next = -1; // 暂时置为-1，后续可能更新
        }

        current = next_addr; // 移动到下一个节点
    }

    // 输出去重后的链表
    current = new_head;
    while (current != -1) {
        if (nodes[current].next == -1) {
            printf("%05d %d -1\n", current, nodes[current].key);
        } else {
            printf("%05d %d %05d\n", current, nodes[current].key, nodes[current].next);
        }
        current = nodes[current].next;
    }

    // 输出被删除的链表
    current = del_head;
    while (current != -1) {
        if (nodes[current].next == -1) {
            printf("%05d %d -1\n", current, nodes[current].key);
        } else {
            printf("%05d %d %05d\n", current, nodes[current].key, nodes[current].next);
        }
        current = nodes[current].next;
    }

    return 0;
}
```
