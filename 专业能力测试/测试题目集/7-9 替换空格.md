# 题目

输入字符串，把空格换成A

### 输入格式:

一个字符串

### 输出格式:

替换后的字符串

### 输入样例:

```in
abc def
```

### 输出样例:

```out
abcAdef
```
# 解答

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string s;
    // 读取包含空格的完整字符串
    getline(cin, s);
    
    // 遍历字符串，将空格替换为'A'
    for (int i = 0; i < s.size(); ++i) {
        if (s[i] == ' ') {
            s[i] = 'A';
        }
    }
    
    // 输出替换后的字符串
    cout << s << endl;
    
    return 0;
}
    
```
