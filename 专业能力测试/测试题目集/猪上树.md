# 题目
已知Pig类是MagicPig类的父类，根据以下主函数完成上述两个类的定义，以实现规定的输出。不允许改变主函数的内容。

### 裁判测试程序样例：

```
#include<iostream>
using namespace std;
/* 请在这里填写答案 */

int main()
{
    Pig *p1 = new Pig; 
    Pig *p2 = new MagicPig; 
    cout << endl; 
    
    p1->climb(); 
    p2->climb(); 
    cout << endl; 
    
    delete p1; 
    delete p2;  
    return 0;
}
```

### 输入样例：

无输入

### 输出样例：

```out
Pig 申请了空间...
Pig 申请了空间...
MagicPig 申请了空间...

我只是个平凡的猪猪。
我能上树了耶！

Pig 释放了空间...
MagicPig 释放了空间...
Pig 释放了空间...
```
# 解答
```cpp
// 定义Pig类
class Pig {
public:
    // 构造函数，输出申请空间信息
    Pig() {
        cout << "Pig 申请了空间..." << endl;
    }
    // 虚析构函数，输出释放空间信息
    virtual ~Pig() {
        cout << "Pig 释放了空间..." << endl;
    }
    // 虚函数climb，定义平凡猪的行为
    virtual void climb() {
        cout << "我只是个平凡的猪猪。" << endl;
    }
};

// 定义MagicPig类，继承自Pig类
class MagicPig : public Pig {
public:
    // 构造函数，先调用父类构造函数，再输出自身申请空间信息
    MagicPig() {
        cout << "MagicPig 申请了空间..." << endl;
    }
    // 析构函数，输出自身释放空间信息，之后自动调用父类析构函数
    ~MagicPig() {
        cout << "MagicPig 释放了空间..." << endl;
    }
    // 重写climb函数，定义魔法猪的行为
    void climb() override {
        cout << "我能上树了耶！" << endl;
    }
};
```
