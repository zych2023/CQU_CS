# 题目
一天，博神搞来了一堆火柴棒，并用他们摆成一个个数字就像下面这样。

![](https://images.ptausercontent.com/217)  
这时马老板走了过来，他突然意识到嘲讽波神的机会来了。他对波神说：给你n根火柴棍，你可以拼出多少个形如“A+B=C”的等式？等式中的A、B、C是用火柴棍拼出的整数（若该数非零，则最高位不能是0）  
注意：

1. 加号与等号各自需要两根火柴棍
2. 如果A≠B，则A+B=C与B+A=C视为不同的等式（A、B、C>=0）
3. n根火柴棍必须全部用上

### 输入格式:

一个整数，表示火柴棒的个数。保证火柴棒的数目不会超过24

### 输出格式:

共一行，表示能拼成的不同等式的数目

### 输入样例:

```in
14
```

### 输出样例:

```out
2
```

# 解答
```cpp
#include <iostream>
using namespace std;

// 每个数字需要的火柴棒数量
int match[] = {6, 2, 5, 5, 4, 5, 6, 3, 7, 6};

// 计算一个数需要的火柴棒总数
int count_matches(int num) {
    if (num == 0) {
        return match[0];
    }
    int cnt = 0;
    while (num > 0) {
        cnt += match[num % 10];
        num /= 10;
    }
    return cnt;
}

int main() {
    int n;
    cin >> n;
    
    // 加号和等号共需4根火柴，剩余的用于数字
    int total = n - 4;
    if (total < 0) {
        cout << 0 << endl;
        return 0;
    }
    
    int count = 0;
    // 遍历可能的A和B的值（范围根据总火柴数估算，确保覆盖所有可能）
    for (int A = 0; A <= 1000; ++A) {
        int a_matches = count_matches(A);
        if (a_matches > total) {
            continue; // A的火柴数已超过剩余，无需继续增大A
        }
        for (int B = 0; B <= 1000; ++B) {
            int b_matches = count_matches(B);
            if (a_matches + b_matches > total) {
                continue; // A+B的火柴数已超过剩余，无需继续增大B
            }
            int C = A + B;
            int c_matches = count_matches(C);
            // 检查总火柴数是否等于剩余
            if (a_matches + b_matches + c_matches == total) {
                count++;
            }
        }
    }
    
    cout << count << endl;
    return 0;
}
```
