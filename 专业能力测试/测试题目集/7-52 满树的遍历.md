# 题目
一棵“k 阶满树”是指树中所有非叶结点的度都是 k 的树。给定一棵树，你需要判断其是否为 k 阶满树，并输出其前序遍历序列。

注：树中结点的度是其拥有的子树的个数，而树的度是树内各结点的度的最大值。

### 输入格式：

输入首先在第一行给出一个正整数 n（≤105），是树中结点的个数。于是设所有结点从 1 到 n 编号。  
随后 n 行，第 i 行（1≤i≤n）给出第 i 个结点的父结点编号。根结点没有父结点，则对应的父结点编号为 `0`。题目保证给出的是一棵合法多叉树，只有唯一根结点。

### 输出格式：

首先在一行中输出该树的度。如果输入的树是 k 阶满树，则加 1 个空格后输出 `yes`，否则输出 `no`。最后在第二行输出该树的前序遍历序列，数字间以 1 个空格分隔，行首尾不得有多余空格。  
注意：兄弟结点按编号升序访问。

### 输入样例 1：

```in
7
6
5
5
6
6
0
5
```

### 输出样例 1：

```out
3 yes
6 1 4 5 2 3 7
```

### 输入样例 2：

```in
7
6
5
5
6
6
0
4
```

### 输出样例 2：

```out
3 no
6 1 4 7 5 2 3
```

# 解答
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <stack>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<vector<int>> children(n + 1);  // 1-based indexing
    int root = -1;

    // 读取父节点信息并构建树
    for (int i = 1; i <= n; ++i) {
        int p;
        cin >> p;
        if (p == 0) {
            root = i;
        } else {
            children[p].push_back(i);
        }
    }

    // 对子节点按编号升序排序
    for (int i = 1; i <= n; ++i) {
        sort(children[i].begin(), children[i].end());
    }

    // 计算每个节点的度和树的度
    vector<int> degrees(n + 1);
    int tree_degree = 0;
    for (int i = 1; i <= n; ++i) {
        degrees[i] = children[i].size();
        if (degrees[i] > tree_degree) {
            tree_degree = degrees[i];
        }
    }

    // 判断是否为k阶满树
    bool is_full = true;
    for (int i = 1; i <= n; ++i) {
        if (degrees[i] != 0 && degrees[i] != tree_degree) {
            is_full = false;
            break;
        }
    }

    // 前序遍历（使用迭代法避免递归栈溢出）
    vector<int> preorder;
    stack<int> st;
    st.push(root);

    while (!st.empty()) {
        int u = st.top();
        st.pop();
        preorder.push_back(u);

        // 逆序压栈，保证弹出时按升序访问子节点
        for (auto it = children[u].rbegin(); it != children[u].rend(); ++it) {
            st.push(*it);
        }
    }

    // 输出结果
    cout << tree_degree << " " << (is_full ? "yes" : "no") << "\n";
    for (size_t i = 0; i < preorder.size(); ++i) {
        if (i > 0) {
            cout << " ";
        }
        cout << preorder[i];
    }
    cout << "\n";

    return 0;
}
```
