# 题目

现在需要输入一组学生的姓名和成绩，然后输出这些学生的姓名和等级。

输入时，首先要输入学生数（正整数）N。接着输入N组学生成绩，每组成绩包括两项：第一项是学生姓名，第二项是学生的成绩（整数）。

输出时，依次输出各个学生的序号（从1开始顺序编号），学生姓名，成绩等级（不小于60为PASS，否则为FAIL）

### 函数接口定义：

```c++
面向Student类对象的流插入和流提取运算符
```

### 裁判测试程序样例：

```c++
#include <iostream>
#include <string>
using namespace std;

/* 请在这里填写答案 */

int main(){
    int i, repeat;
    Student st;
    cin>>repeat;
    for(i=0;i<repeat;i++){
        cin>>st;
        cout<<st<<endl;
    }
    return 0;
}
```

### 输入样例：

```in
3
Li 75
Zhang 50
Yang 99
```

### 输出样例：

```out
1. Li PASS
2. Zhang FAIL
3. Yang PASS
```

# 解答

```cpp
class Student {
private:
    string name;       // 学生姓名
    int score;         // 学生成绩
    int no;            // 学生序号（从1开始）
    static int count;  // 静态成员：记录输入的学生总数（用于生成序号）
public:
    // 友元声明：允许流运算符访问私有成员
    friend istream& operator>>(istream& is, Student& s);
    friend ostream& operator<<(ostream& os, const Student& s);
};

// 初始化静态成员（必须在类外初始化）
int Student::count = 0;

// 重载>>运算符：读取姓名和成绩，并自动生成序号
istream& operator>>(istream& is, Student& s) {
    is >> s.name >> s.score;
    Student::count++;  // 每读取一个学生，计数+1
    s.no = Student::count;  // 序号 = 当前计数（自动从1开始）
    return is;
}

// 重载<<运算符：输出序号、姓名和等级
ostream& operator<<(ostream& os, const Student& s) {
    os << s.no << ". " << s.name << " " 
       << (s.score >= 60 ? "PASS" : "FAIL");
    return os;
}
```
