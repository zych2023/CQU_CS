# 题目
一个Time类，数据成员有时、分、秒。要求模拟秒表，每次走一秒，满60秒进位，秒又从零开始计数。满60分进位，分又从零开始计数。输出时、分和秒的值。（使用重载++运算符实现）

### 时间类定义：

```c++
class MyTime
```

### 测试程序样例：

```c++
/* 请在这里填写答案 */

int main()
{
    MyTime t1,t2(23,59,59),t3;
    cin>>t3;
    ++t1;
    cout<<t1<<endl;
    ++t2;
    cout<<t2<<endl;
    ++t3;
    cout<<t3<<endl;
    return 0;
}
```

### 输入样例：

```in
12 35 59
```

### 输出样例：

```out
0:0:1
0:0:0
12:36:0
```

# 解答
```cpp
#include <iostream>
using namespace std;

class MyTime {
private:
    int hour;
    int minute;
    int second;
public:
    // 默认构造函数，初始时间为0:0:0
    MyTime() : hour(0), minute(0), second(0) {}
    
    // 带参数的构造函数，初始化时、分、秒
    MyTime(int h, int m, int s) : hour(h), minute(m), second(s) {}
    
    // 重载前缀++运算符，实现时间加1秒
    MyTime& operator++() {
        second++;
        // 处理秒进位
        if (second >= 60) {
            second -= 60;
            minute++;
            // 处理分进位
            if (minute >= 60) {
                minute -= 60;
                hour++;
                // 处理时进位（满24小时重置为0）
                if (hour >= 24) {
                    hour -= 24;
                }
            }
        }
        return *this;
    }
    
    // 友元函数，重载输出运算符
    friend ostream& operator<<(ostream& os, const MyTime& t) {
        os << t.hour << ":" << t.minute << ":" << t.second;
        return os;
    }
    
    // 友元函数，重载输入运算符
    friend istream& operator>>(istream& is, MyTime& t) {
        is >> t.hour >> t.minute >> t.second;
        return is;
    }
};
```
