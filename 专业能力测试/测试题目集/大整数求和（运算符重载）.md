# 题目

BigInt类表示不超过100位的无符号大整数。试重载>>，<<和+，以支持无符号大整数的输入、输出与求和（假设结果仍是一个不超过100位的无符号大整数）。

### 重载面向BigInt类对象的运算符：

```c++
>>
<<
+
```

### 裁判测试程序样例：

```c++
#include <iostream>
#include <string>
using namespace std;

/* 请在这里填写答案 */

int main(){
    BigInt a, b, c;
    cin>>a>>b;
   c=a+b;
    cout<<a<<"+"<<b<<"="<<c<<endl;
    return 0;
}
```

### 输入样例：

```in
123456789
987654321
```

### 输出样例：

```out
123456789+987654321=1111111110
```
# 解答
```cpp
class BigInt {
private:
    string num;  // 字符串存储大整数（如"123456789"表示123456789）
public:
    // 默认构造函数
    BigInt() : num("") {}
    // 带字符串参数的构造函数（用于创建加法结果）
    BigInt(const string& s) : num(s) {}

    // 2. 重载+运算符：实现大整数加法
    BigInt operator+(const BigInt& other) const {
        string a = this->num;
        string b = other.num;
        string res;
        int carry = 0;  // 进位标志
        int i = a.size() - 1;  // 指向a的最低位（字符串末尾）
        int j = b.size() - 1;  // 指向b的最低位（字符串末尾）

        // 从低位到高位计算，处理所有位和进位
        while (i >= 0 || j >= 0 || carry > 0) {
            int digitA = (i >= 0) ? (a[i] - '0') : 0;
            int digitB = (j >= 0) ? (b[j] - '0') : 0;
            int sum = digitA + digitB + carry;

            res.insert(res.begin(), (sum % 10) + '0');  // 个位插入结果开头
            carry = sum / 10;  // 更新进位（十位）

            i--;
            j--;
        }
        return BigInt(res);
    }

    // 3. 友元声明：允许>>和<<访问私有成员num
    friend istream& operator>>(istream& in, BigInt& obj);
    friend ostream& operator<<(ostream& out, const BigInt& obj);
};

// 4. 重载>>运算符：输入大整数（字符串形式）
istream& operator>>(istream& in, BigInt& obj) {
    in >> obj.num;
    return in;
}

// 5. 重载<<运算符：输出大整数（字符串形式）
ostream& operator<<(ostream& out, const BigInt& obj) {
    out << obj.num;
    return out;
}
```
