# 题目
“侧影”就是从左侧或者右侧去观察物体所看到的内容。例如上图中男生的侧影是从他右侧看过去的样子，叫“右视图”；女生的侧影是从她左侧看过去的样子，叫“左视图”。

520 这个日子还在打比赛的你，也就抱着一棵二叉树左看看右看看了……

我们将二叉树的“侧影”定义为从一侧能看到的所有结点从上到下形成的序列。例如下图这棵二叉树，其右视图就是 { 1, 2, 3, 4, 5 }，左视图就是 { 1, 6, 7, 8, 5 }。

![fig.JPG](https://images.ptausercontent.com/6c932eac-05f4-4d3f-8490-4c5f2b1c457e.JPG)

于是让我们首先通过一棵二叉树的中序遍历序列和后序遍历序列构建出一棵树，然后你要输出这棵树的左视图和右视图。

### 输入格式：

输入第一行给出一个正整数 N (≤20)，为树中的结点个数。随后在两行中先后给出树的中序遍历和后序遍历序列。树中所有键值都不相同，其数值大小无关紧要，都不超过 **int** 的范围。

### 输出格式：

第一行输出右视图，第二行输出左视图，格式如样例所示。

### 输入样例：

```in
8
6 8 7 4 5 1 3 2
8 5 4 7 6 3 2 1
```

### 输出样例：

```out
R: 1 2 3 4 5
L: 1 6 7 8 5
```

# 解答
```cpp
#include<bits/stdc++.h>
using namespace std;

struct TreeNode{
    int val;
    TreeNode*left;
    TreeNode*right;
    TreeNode(int v):val(v),left(nullptr),right(nullptr){}
};
vector<int>zh,ho;

TreeNode* bT(int root,int s,int end){
    if(s>end)return nullptr;
    int i=s;
    while(zh[i]!=ho[root])i++;
    TreeNode*node=new TreeNode(zh[i]);
    node->left=bT(root - (end - i) - 1,s,i-1);//(i-1,s,i-1);
    node->right=bT(root-1,i+1,end);//(root-1,i,end-1);
    return node;
}

vector<vector<int>>ceng(TreeNode*root){
    queue<TreeNode*>q;
    vector<vector<int>>anss;
    if(!root)return anss;
    q.push(root);
    while(!q.empty()){
        vector<int>ans;
        int size=q.size();  // 当前层的节点数
        for(int i=0;i<size;i++){
            TreeNode*cur=q.front();
            q.pop();
            ans.push_back(cur->val);
            if(cur->left)q.push(cur->left);  // 左孩子入队
            if(cur->right)q.push(cur->right);  // 右孩子入队
        }
        anss.push_back(ans);  // 保存当前层
    }
    return anss;
}

void sleft(vector<vector<int>>ce){
    int n=ce.size();
    cout<<"R:";
    for(int i=0;i<n;i++){
        int m=ce[i].size();
        cout<<' '<<ce[i][m-1];
    }
    cout<<endl;
}

void sright(vector<vector<int>>ce){
    int n=ce.size();
    cout<<"L:";
    for(int i=0;i<n;i++){
        // int m=ce[i].size();
        cout<<' '<<ce[i][0];
    }
    cout<<endl;
}

int main(){
    int N,nn;
    cin>>N;
    for(int i=0;i<N;i++){
        cin>>nn;
        zh.push_back(nn);
    }
    for(int i=0;i<N;i++){
        cin>>nn;
        ho.push_back(nn);
    }
    TreeNode*node=bT(N-1,0,N-1);
    vector<vector<int>>anssss=ceng(node);
    
    sleft(anssss);
    sright(anssss);
    return 0;
}
```
