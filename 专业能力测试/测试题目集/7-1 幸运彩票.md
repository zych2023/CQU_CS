# 题目
彩票的号码有 6 位数字，若一张彩票的前 3 位上的数之和等于后 3 位上的数之和，则称这张彩票是幸运的。本题就请你判断给定的彩票是不是幸运的。

### 输入格式：

输入在第一行中给出一个正整数 N（≤ 100）。随后 N 行，每行给出一张彩票的 6 位数字。

### 输出格式：

对每张彩票，如果它是幸运的，就在一行中输出 `You are lucky!`；否则输出 `Wish you good luck.`。

### 输入样例：

```in
2
233008
123456
```

	### 输出样例：

```out
You are lucky!
Wish you good luck.
```

# 解答

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    int n;
    cin >> n;
    
    // 处理每张彩票
    for (int i = 0; i < n; i++) {
        string lottery;
        cin >> lottery;
        
        // 计算前3位数字之和
        int sumFirstHalf = 0;
        for (int j = 0; j < 3; j++) {
            sumFirstHalf += lottery[j] - '0'; // 将字符转换为数字
        }
        
        // 计算后3位数字之和
        int sumSecondHalf = 0;
        for (int j = 3; j < 6; j++) {
            sumSecondHalf += lottery[j] - '0';
        }
        
        // 判断并输出结果
        if (sumFirstHalf == sumSecondHalf) {
            cout << "You are lucky!" << endl;
        } else {
            cout << "Wish you good luck." << endl;
        }
    }
    
    return 0;
}

```
