# 题目
补充下列代码，使得程序的输出为：  
  
A:3  
  
A:15  
  
B:5  
  
3  
  
15  
  
5

### 类和函数接口定义：

```c++
参见裁判测试程序样例中的类和函数接口。
```

### 裁判测试程序样例：

```c++
#include <iostream>
using namespace std;
class CMyClassA {
    int val;
public:
    CMyClassA(int);
    void virtual print();
};
CMyClassA::CMyClassA(int arg) {
    val = arg;
    printf("A:%d\n", val);
}
void CMyClassA::print() {
    printf("%d\n", val);
    return;
}

/* 在这里填写代码 */

int main(int argc, char** argv) {
    CMyClassA a(3), *ptr;
    CMyClassB b(5);
    ptr = &a;
    ptr->print();
    a = b;
    a.print();
    ptr = &b;
    ptr->print();
    return 0;
}
```

### 输入样例：

```in
None
```

### 输出样例：

```out
A:3
A:15
B:5
3
15
5
```

# 解答
```cpp
class CMyClassB : public CMyClassA {
private:
    int val_b; // 存储B类自身的数值
public:
    // 构造函数，初始化基类为15，自身数值为传入的arg
    CMyClassB(int arg) : CMyClassA(15), val_b(arg) {
        printf("B:%d\n", val_b);
    }
    // 重写print函数，输出B类的数值
    void print() override {
        printf("%d\n", val_b);
    }
};
```
