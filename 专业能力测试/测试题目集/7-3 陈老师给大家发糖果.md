# 题目
程序员节日快要到了，陈老师正在为大家准备礼物，就是最可口的糖果。陈老师一共拥有N块糖果，为了显得糖果很多，陈老师把糖果排成一列，然后每一块糖果都设置了编号，编号按照摆放顺序设置为从1到N的自然数。  
陈老师发糖果的时候，不想直接按顺序一块挨一块的发，他先从1号开始，按照编号顺序，每隔M块糖拿一块发给程序员们……按照上述规则，从开头到结尾拿完一轮之后，陈老师会把剩余的糖果再重新摆成连续的一列，继续开启新的一轮发糖果……直到最后一块糖也发出去。  
请问陈老师几轮能够发完所有的糖果？

### 输入格式:

输入两个正整数N和M，分别代表陈老师最初拥有的糖数量，以及他拿糖果时候的间隔数M。  
1<= N <=1000000000 ,   1<= M <=5

### 输出格式:

输出一个整数，代表陈老师几轮能够发完所有糖果。

### 输入样例:

在这里给出一组输入。例如：

```in
8 2
```

### 输出样例:

在这里给出相应的输出。例如：

```out
5
```

### 补充说明:

陈老师最初有8块糖果，每隔2个发一个。  
第1轮发编号为1、4、7 的糖，  
第轮发编号为 2、6 的糖，  
第三轮发编号为 3 的糖，  
第四轮发编号为 5 的糖，  
第五轮发拿编号为 8 的糖。  
1 2 3 4 5 6 7 8     2 3 5 6 8      3 5 8     5 8     8

# 解答

```cpp
#include <iostream>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    
    int step = M + 1;  // 实际步长为间隔数+1
    int long rounds = 0;
    int remaining = N;
    
    while (remaining > 0) {
        rounds++;
        // 计算本轮发放的糖果数量（向上取整）
        int take = (remaining + step - 1) / step;
        remaining -= take;
    }
    
    cout << rounds << endl;
    return 0;
}
```
