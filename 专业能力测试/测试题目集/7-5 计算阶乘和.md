# 题目
对于给定的正整数N，需要你计算 S=1!+2!+3!+...+N!。

### 输入格式：

输入在一行中给出一个不超过10的正整数N。

### 输出格式：

在一行中输出S的值。

### 输入样例：

```in
3
```

### 输出样例：

```out
9
```

# 解答

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;  // 读取输入的正整数N
    
    int sum = 0;       // 用于存储阶乘之和
    int factorial = 1; // 用于存储当前数的阶乘
    
    // 计算1!到N!的和
    for (int i = 1; i <= N; ++i) {
        factorial *= i;  // 计算当前i的阶乘
        sum += factorial; // 将当前阶乘加入总和
    }
    
    cout << sum << endl; // 输出结果
    
    return 0;
}
    
```
