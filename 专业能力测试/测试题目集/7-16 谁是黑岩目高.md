# 题目
追求是个四斋，这天他看番看到了《我的可爱对黑岩目高不管用》。他觉得男主太牛逼了，面对这么可爱的女主都不为所动。所以他想请你找出哪些人是男主黑岩目高。

已知：

1. 黑岩目高在让自己冷静下来时会说 “shinto mekkaku”，但他有时候着急了会大小写混合。

2. 黑岩目高可能随时需要冷静，并且冷静前后仍然有可能说其他内容。

如果这个人是黑岩目高，请你输出 `Medaka Kuroiwa`；

如果不是，请输出 `Bonnjinn`。

### 输入格式:

第一行输入一个正整数 T (T≤103)。 接下来 T 行，每行一个长度不超过1000的字符串。

### 输出格式:

对于每个字符串，输出一个字符串表示这个人的身份。

### 输入样例:

```in
2
shinto mekkaku
hello
```

### 输出样例:

```out
Medaka Kuroiwa
Bonnjinn
```

# 解答

```cpp
#include <iostream>
#include <string>
#include <cctype>

using namespace std;

// 将字符串转换为全小写
string toLower(const string &s) {
    string res;
    for (char c : s) {
        res += tolower(c);
    }
    return res;
}

int main() {
    int T;
    cin >> T;
    cin.ignore(); // 忽略输入T后的换行符，避免影响后续getline读取
    
    string target = "shinto mekkaku";
    string lowerTarget = toLower(target);
    
    for (int i = 0; i < T; ++i) {
        string s;
        getline(cin, s);
        string lowerS = toLower(s);
        
        if (lowerS.find(lowerTarget) != string::npos) {
            cout << "Medaka Kuroiwa" << endl;
        } else {
            cout << "Bonnjinn" << endl;
        }
    }
    
    return 0;
}v
```
