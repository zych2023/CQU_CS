# 题目

输出给定年份区间（该区间不超出[1900, 2999]）之内的所有闰年，输出时每个年份占6列（右对齐），每5个闰年换一行。闰年的判断条件是该年年份能被4整除但不能被100整除，或者能被400整除。

### 输入格式:

输入起始年份begin以及结束年份end，中间用一个空格间隔

### 输出格式:

输出[begin, end]区间内所有的闰年。  
每个年份占6列（右对齐），每5个闰年换一行。

### 输入样例:

```in
1980 2024
```

### 输出样例:

```out
  1980  1984  1988  1992  1996
  2000  2004  2008  2012  2016
  2020  2024
```

# 解答

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

// 判断是否为闰年
bool isLeapYear(int year) {
    return (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
}

int main() {
    int begin, end;
    cin >> begin >> end;
    
    int count = 0; // 记录已输出的闰年数量，用于控制换行
    for (int year = begin; year <= end; ++year) {
        if (isLeapYear(year)) {
            // 输出年份，占6列右对齐
            cout << setw(6) << year;
            count++;
            // 每5个闰年换一行
            if (count % 5 == 0) {
                cout << endl;
            }
        }
    }
    // 如果最后一行不足5个，补充换行
    if (count % 5 != 0) {
        cout << endl;
    }
    
    return 0;
}
```
