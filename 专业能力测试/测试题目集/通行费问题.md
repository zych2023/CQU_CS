# 题目
城市A、B之间的高速通行费标准因车型不同而异。其中，小型汽车‌通行费标准是每公里0.6元‌；大型汽车通行费标准是每公里1.5元。春节假期小型汽车‌可以免费通行。

现对一组汽车通行记录，计算其‌高速通行费。一条汽车通行记录包含3个数据项：第1项是汽车类型（整数1表示小型汽车，整数2表示大型汽车）；第2项是通行时间（整数1表示在春节假期，整数0表示不在春节假期）；第3项是通行距离（浮点数表示，单位：公里）。用浮点数表示高速通行费，输出结果保留2位小数。

请根据裁判测试程序，完善TravelRecord类的子类CarTravelRecord、TruckTravelRecord类的定义。其中CarTravelRecord类表示小型汽车‌通行记录；TruckTravelRecord类表示大型汽车通行记录。

# 解答
```cpp
// 小型汽车通行记录类
class CarTravelRecord : public TravelRecord {
public:
    // 构造函数，调用基类构造函数初始化
    CarTravelRecord(int inHoliday, double distance) : TravelRecord(inHoliday, distance) {}
    
    // 计算通行费：假期免费，非假期0.6元/公里
    double toll() override {
        if (inHoliday == 1) {
            return 0.0;
        } else {
            return distance * 0.6;
        }
    }
};

// 大型汽车通行记录类
class TruckTravelRecord : public TravelRecord {
public:
    // 构造函数，调用基类构造函数初始化
    TruckTravelRecord(int inHoliday, double distance) : TravelRecord(inHoliday, distance) {}
    
    // 计算通行费：1.5元/公里，不受假期影响
    double toll() override {
        return distance * 1.5;
    }

```
