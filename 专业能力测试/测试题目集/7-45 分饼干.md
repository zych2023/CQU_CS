# 题目
易老师购买了一盒饼干，盒子中一共有k块饼干，但是数字k有些数位变得模糊了，看不清楚数字具体是多少。易老师需要你帮忙把这k块饼干平分给n个小朋友，易老师保证这盒饼干能平分给n个小朋友。现在需要计算出k有多少种可能的数值。

### 输入格式:

输入包括两行，第1行为盒子上的数值k，模糊的数位用X表示，长度小于18（ 可能有多个模糊的数位），第2行为小朋友的人数n。

### 输出格式:

输出k可能的数值种数，保证至少为1。

### 输入样例:

在这里给出一组输入。例如：

```in
9999999999999X
3
```

### 输出样例:

在这里给出相应的输出。例如：

```out
4
```

# 解答
```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    string k;
    int n;
    cin >> k >> n;
    int len = k.size();
    
    // dp[pos][rem]表示处理完前pos位后，余数为rem的方案数
    vector<vector<long long>> dp(len + 1, vector<long long>(n, 0));
    dp[0][0] = 1;  // 初始状态：处理0位，余数0，1种方案
    
    for (int pos = 0; pos < len; ++pos) {
        for (int rem = 0; rem < n; ++rem) {
            if (dp[pos][rem] == 0) {
                continue;  // 该状态无方案，跳过
            }
            
            // 确定当前位可能的数字
            char c = k[pos];
            vector<int> possible_digits;
            if (c == 'X') {
                // X可以是0-9
                for (int d = 0; d <= 9; ++d) {
                    possible_digits.push_back(d);
                }
            } else {
                // 固定数字
                possible_digits.push_back(c - '0');
            }
            
            // 处理每个可能的数字，更新下一位状态
            for (int d : possible_digits) {
                int new_rem = (rem * 10 + d) % n;
                dp[pos + 1][new_rem] += dp[pos][rem];
            }
        }
    }
    
    cout << dp[len][0] << endl;
    
    return 0;
} 	
```
