# 题目
假设一个购物车有固定的容积V，需要往里面装入一个价格最低的商品。

现有若干商品，商品信息包括：商品名、商品体积、商品价格。

请计算并求出该购物袋能装下（即：商品体积<=购物车容积V）的价格最低的商品信息。

### 输入格式:

首先输入一个整数V代表购物车的容积，和一个整数n代表商品数量；

然后输入n个商品的信息，商品信息包括：商品名(字符长度小于20)、商品体积(int类型)、商品价格(double类型)。

### 输出格式:

输出购物车能装下的价格最低的商品信息包括：商品名 体积 价格。价格的小数点后保留2位。每个信息之间一个空格间隔，末尾没有空格。

### 输入样例:

  

```in
10 5
apple 10 6.99
banana 11 5.99
milk 8 5.99
cola 10 4.99
bread 12 3.99
```

### 输出样例:

  

```out
cola 10 4.99
```

# 解答
```cpp
#include <iostream>
#include <iomanip>
#include <string>
using namespace std;

// 定义商品结构体
struct Product {
    string name;   // 商品名
    int volume;    // 商品体积
    double price;  // 商品价格
};

int main() {
    int V, n;
    cin >> V >> n;

    Product minProduct;
    double minPrice = 1e18;  // 初始化为一个很大的数

    for (int i = 0; i < n; ++i) {
        Product p;
        cin >> p.name >> p.volume >> p.price;

        // 筛选出能装下的商品
        if (p.volume <= V) {
            // 更新最低价格商品
            if (p.price < minPrice) {
                minPrice = p.price;
                minProduct = p;
            }
        }
    }

    // 输出结果，价格保留两位小数
    cout << minProduct.name << " " << minProduct.volume << " " 
         << fixed << setprecision(2) << minProduct.price << endl;

    return 0;
}
```
