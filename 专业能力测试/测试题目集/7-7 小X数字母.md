# 题目

小 X 喜欢研究字符串。

这天，小 X 随手在草稿纸上写下了一个大写字母字符串。此时，班长把成绩报告单发到了每位同学的手中。小 X 看到自己每门都是 `A` （优秀），非常高兴，灵光一闪想到一个问题： 在刚刚写下的字符串中，字母 `A` 最多连续出现了多少次呢？

小 X 立刻数了起来，但这个字符串实在是太长了，希望你帮帮他。

### 输入格式:

第一行一个字符串。（长度<=30）

### 输出格式:

第一行包含一个整数，表示该字符串中字母 `A` 最多连续出现的次数。

### 输入样例:

```in
BAACA
```

### 输出样例:

```out
2
```

# 解答

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string s;
    cin >> s;  // 读取输入的字符串
    
    int max_count = 0;      // 记录最大连续A的数量
    int current_count = 0;  // 记录当前连续A的数量
    
    // 遍历字符串中的每个字符
    for (char c : s) {
        if (c == 'A') {
            // 如果是A，当前计数器加1
            current_count++;
            // 更新最大记录（如果当前连续数量更大）
            if (current_count > max_count) {
                max_count = current_count;
            }
        } else {
            // 如果不是A，重置当前计数器
            current_count = 0;
        }
    }
    
    // 输出结果
    cout << max_count << endl;
    
    return 0;
}
    
```
