# 题目
定义猫和老鼠：Cat与Mouse两个类，二者都有weight属性，定义二者的一个友元函数totalweight()，计算二者的重量和。

### 裁判测试程序样例：

```
#include <iostream>
using namespace std;

/* 请在这里填写答案 */

int main()
{
  int w1,w2;
  cin>>w1>>w2;
  Cat tom(w1);
  Mouse jerry(w2);
  cout<<totalWeight(tom,jerry)<<endl;
}
```

### 输入样例：

```in
100 200
```

### 输出样例：

```out
300
```

# 解答
```cpp
// 提前声明Mouse类，以便Cat类中友元函数的声明
class Mouse;

// 定义Cat类
class Cat {
private:
    int weight; // 猫的重量
public:
    // 构造函数，初始化猫的重量
    Cat(int w) : weight(w) {}
    // 声明totalWeight为友元函数
    friend int totalWeight(const Cat& cat, const Mouse& mouse);
};

// 定义Mouse类
class Mouse {
private:
    int weight; // 老鼠的重量
public:
    // 构造函数，初始化老鼠的重量
    Mouse(int w) : weight(w) {}
    // 声明totalWeight为友元函数
    friend int totalWeight(const Cat& cat, const Mouse& mouse);
};

// 友元函数实现，计算猫和老鼠的重量和
int totalWeight(const Cat& cat, const Mouse& mouse) {
    return cat.weight + mouse.weight;
}
```
