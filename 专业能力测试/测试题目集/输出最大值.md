# 题目
根据给定的程序，写成相关的成员函数，完成指定功能。

### 函数接口定义：

```c++
定义max函数，实现输出最高成绩对应的学号以及最高成绩值。
```

### 裁判测试程序样例：

```c++
#include <iostream>
using namespace std;
class Student
 {public:
   Student(int n,float s):num(n),score(s){}
   int num;
   float score;
 };

int main()
{Student stud[5]={
  Student(101,78.5),Student(102,85.5),Student(103,98.5),
  Student(104,100.0),Student(105,95.5)};
 void max(Student* );
 Student *p=&stud[0];
 max(p);
 return 0; 
 }
/* 请在这里填写答案 */
```

### 输入样例：

```in
无
```

### 输出样例：

```out
104 100
```
# 解答
```cpp
void max(Student* p) {
    int max_num = p[0].num;       // 初始化最高成绩对应的学号为第一个学生的学号
    float max_score = p[0].score; // 初始化最高成绩为第一个学生的成绩
    
    // 遍历数组中剩余的4个学生（总共有5个学生）
    for (int i = 1; i < 5; ++i) {
        if (p[i].score > max_score) {
            max_score = p[i].score; // 更新最高成绩
            max_num = p[i].num;     // 更新最高成绩对应的学号
        }
    }
    
    // 输出最高成绩对应的学号和成绩
    cout << max_num << " " << max_score << endl;
}
```
