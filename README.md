# CQU_CS

重庆大学计算机科学与技术专业部分课程存档。

包含实验代码、实验报告、期末复习资料等，部分资料由学长学姐传承整理，希望能对后来的同学有所帮助。

## 课程目录

| 课程 | 内容 |
|------|------|
| 编译原理 | SysY 编译器实验（词法分析→RISC-V 代码生成）、毕昇编译器优化实验 |
| 并行与分布式计算 | OpenMP/MPI 实验、PageRank 并行计算课设 |
| 自然语言处理 | Viterbi 算法、古诗生成、Qwen2 LoRA 微调实验 |
| 计算机组成原理 | MIPS CPU Verilog 实验、教材习题答案 |
| 操作系统 | 实验报告、期末复习资料 |
| 数据库系统 | 期末复习资料、历年试题 |
| 计算机网络 | 实验报告、各章复习笔记、历年试题 |
| 软件工程 | 期末复习资料 |
| 专业能力测试 | PAT 风格 C++ 编程题集 |
| 开卷打印小工具 | PPT/PDF 开卷考试打印辅助工具 |
