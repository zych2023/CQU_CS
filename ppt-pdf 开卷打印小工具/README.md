# PPT-PDF 缩印 & 合并工具

批量处理 PPT/PDF 文件的桌面小工具，支持多合一缩印、PDF 合并、PDF 拆分。
powered by zych2023

## 功能

| 功能 | 说明 |
|------|------|
| **PPT 缩印** | PPTX → 多合一 PDF（2×2 / 3×3 / 4×4 等布局） |
| **PDF 缩印** | PDF → 多合一 PDF（同上，矢量合成，清晰度高） |
| **PDF 合并** | 多个 PDF 按顺序合并为一个 |
| **PDF 拆分** | 逐页拆分 / 按页码范围拆分（如 1-5, 8, 10-15） |

## 清晰度保证

- PPT 通过 **PowerPoint COM** 导出原生矢量 PDF（非截图）
- PDF 缩印使用 PyMuPDF 的 `show_pdf_page()` **矢量合成**，零光栅化
- 图像内容使用双三次插值缩放，JPEG 质量 95%

## 环境要求

- **Windows** 10/11
- **Python 3.10+**（推荐 Anaconda/Miniconda）
- **Microsoft PowerPoint**（PPT 缩印功能需要）
- **PyMuPDF** + **pywin32**

## 安装

```bash
# 1. 激活 conda 环境（或你的 Python 环境）
conda activate base

# 2. 安装依赖
pip install -r requirements.txt

# 依赖清单：
# PyMuPDF>=1.25.0
# pywin32>=308
```

## 使用

```bash
conda activate base
python app.py
```

启动后出现图形界面，三个标签页对应三种功能：

### PPT 缩印
1. 点击「添加文件」选择 `.pptx` 文件
2. 选择布局（如 3×3 = 每页放 9 张幻灯片）
3. 选择输出目录（默认与源文件同目录）
4. 点击「开始缩印」

### PDF 缩印
1. 添加 `.pdf` 文件
2. 选择布局
3. 点击「开始缩印」

### PDF 合并
1. 添加多个 PDF 文件
2. 用「上移/下移」调整顺序
3. 点击「合并为一个 PDF」

### PDF 拆分
1. 选择一个 PDF 文件
2. 填写页码范围（留空 = 逐页拆分）
   - 格式：`1-5, 8, 10-15`
3. 点击「开始拆分」

| 功能 | 输出文件名示例 |
|------|---------------|
| PPT 缩印 | `演示文稿_缩印3x3.pdf` |

## 布局选项

| 布局 | 每页张数 | 适用场景 |
|------|---------|---------|
| 2×2 | 4 | 需要看清内容 |
| 2×3 | 6 | 平衡清晰度和节省纸张 |
| 3×3 | 9 | 最常用缩印密度 |
| 3×4 | 12 | 较紧凑 |
| 4×4 | 16 | 最紧凑，适合速查 |

## 常见问题

**Q: "PowerPoint 转换失败"**
A: 确保 Microsoft PowerPoint 已安装且可正常打开。WPS 不支持。

**Q: 缩印后文字模糊？**
A: 本工具使用矢量合成，理论上不会模糊。如果源 PDF 本身是低分辨率图片，缩印后也会保持原质量。

**Q: 支持 .ppt 旧格式吗？**
A: 不支持。请先用 PowerPoint 另存为 `.pptx` 格式。
