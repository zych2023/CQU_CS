# 实验四操作文档（可复现）

## 环境要求

- 操作系统：Windows 11
- Python 环境：conda (study)
- 依赖包：torch, transformers, datasets, peft, accelerate, matplotlib

## 操作步骤

### 步骤 1：激活环境

```bash
conda activate study
```

### 步骤 2：进入实验目录

```bash
cd D:\Desktop\ex4
```

### 步骤 3：安装依赖（首次运行）

```bash
pip install transformers==4.37.0 datasets==2.16.1 peft==0.6.2 accelerate==0.24.1 matplotlib
```

### 步骤 4：运行完整实验

```bash
python run_local.py
```

运行完成后会生成以下文件：
- `loss_curve.png` - Loss 曲线图
- `comparison_results.json` - 对比结果
- `qwen-poetry-lora-final/` - 微调后的模型

### 步骤 5：快速复现对比结果

```bash
python 快速复现.py
```

运行完成后会生成：
- `复现结果.json` - 新的对比结果

### 步骤 6：生成所有图片

```bash
python 生成图片.py
```

运行完成后会生成：
- `loss_curve.png` - Loss 曲线图
- `params_comparison.png` - 参数对比图
- `lora_ratio.png` - LoRA 参数占比饼图
- `comparison_table.png` - 生成结果对比图
- `training_config.png` - 训练配置总结图

### 步骤 7：查看结果

对比结果在 `复现结果.json` 文件中，格式如下：

```json
[
  {
    "prompt": "床前明月光，",
    "baseline": "Baseline 模型的输出...",
    "lora": "LoRA 微调模型的输出..."
  },
  ...
]
```

## 文件说明

| 文件名 | 说明 |
|--------|------|
| `run_local.py` | 完整实验脚本（训练+对比） |
| `快速复现.py` | 快速复现脚本（只做对比） |
| `生成图片.py` | 生成所有图片 |
| `实验报告.md` | 实验报告模板 |
| `poetry_utf8.txt` | 训练数据（UTF-8编码） |
| `comparison_results.json` | 原始对比结果 |
| `复现结果.json` | 复现的对比结果 |
| `qwen-poetry-lora-final/` | 微调后的模型权重 |

## 注意事项

1. 首次运行需要下载 Qwen2-0.5B-Instruct 模型（约1GB），请确保网络通畅
2. CPU 训练较慢，完整训练约需 50 分钟
3. 快速复现脚本只需加载模型，约需 2-3 分钟
4. 所有脚本已处理中文编码问题，可直接运行
